# 🔥 Python Code Runner

A simple yet powerful web application to run Python code directly in the browser using a custom backend execution engine.

---

## 🚀 Features

* ▶️ Run Python code instantly
* 📄 Live output display in browser
* ❌ Error handling with clear messages
* ⚡ Fast backend execution using Node.js
* 🌐 Clean and minimal user interface

---

## 🧠 System Architecture

```
User → Frontend (HTML/CSS/JS) → Node.js Server → Python Execution → Output → Browser
```

---

## 🛠 Tech Stack

| Layer         | Technology            |
| ------------- | --------------------- |
| Frontend      | HTML, CSS, JavaScript |
| Backend       | Node.js (Express)     |
| Execution     | Python                |
| Communication | REST API              |

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```
git clone https://github.com/YOUR_USERNAME/python-code-runner.git
cd python-code-runner
```

---

### 2️⃣ Install Dependencies

```
npm install
```

---

### 3️⃣ Run the Server

```
npm start
```

---

### 4️⃣ Open in Browser

```
http://localhost:3000
```
